# Tasks - 작업 지시 저장소

이 폴더에는 모든 작업 지시가 저장됩니다.

## 작업 목록

| 파일 | 제목 | 생성일 | 상태 |
|------|------|--------|------|
| template.md | 작업 템플릿 | - | - |

## 새 작업 시작 시

```bash
# 1. 템플릿 복사
cp memory/tasks/template.md memory/tasks/TASK_$(date +%Y%m%d)_이름.md

# 2. 파일 열어서 작성
nano memory/tasks/TASK_$(date +%Y%m%d)_이름.md

# 3. 작업 시작
```

## 작업 상태 라벨

- `[진행중]` - 현재 진행 중인 작업
- `[완료]` - 완료된 작업
- `[보류]` - 일시 중단된 작업
- `[중단]` - 취소된 작업
